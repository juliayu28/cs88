Week 2

Key Takeaways:

This week, we got to use the components we set up last week in order to do some coding and submit our first hw assignment!

General Assignment Setup Process for the Semester:

  1. First, download the files for the lab/hw.
  2. In your Terminal/Git Bash, use cd (the "move into" command) to go to the folder that contains the lab or hw.
  3. However you want, open up the file for the lab/hw in your text editor such as Sublime.
  4. Whenever you want to run the code and test it out, type python filename.py in your Terminal/Git Bash.
Sometimes, we will have questions such as "What would Python Display?" that are answered in your Terminal/Git Bash. These do not have a coding part, so they won't be in your submission.

Command Line vs. Interactive Mode

A lot of students mix the two up, so here are the key differences.

Command Line Interactive Mode
  • $ means you are in it
  • is for typing commands, including cd directoryname and python filename.py
  • Type python or python3 to enter Interactive Mode
  • >>> means you are in it
  • is for typing Python code
  • Type quit() to go back to Command Line
